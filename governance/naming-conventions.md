# Life OS Naming Conventions

**Document ID:** LOS-NAM-001

**Status:** Draft

**Owner:** Architecture Team

---

# Purpose

This document defines the official naming conventions used throughout the Life OS project.

Consistent naming improves readability, maintainability, traceability, and collaboration.

---

# General Rules

- Use English for all technical names.
- Use lowercase for file and directory names.
- Use kebab-case for files and directories.
- Use PascalCase only where required by the implementation language.
- Use meaningful names.
- Avoid abbreviations unless officially documented.
