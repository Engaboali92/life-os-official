# Learning Entity

**Document ID:** LOS-DAT-ENT-012

**Status:** Draft

**Owner:** Data Team

---

# Purpose

The Learning Entity represents educational activities, courses, study sessions, certifications, and knowledge progress managed by Life OS.

---

# Responsibilities

- Store learning records.
- Track study sessions.
- Track certifications.
- Track learning progress.
- Support learning analytics.

---

# Attributes

| Name | Type | Required |
|------|------|----------|
| id | UUID | Yes |
| userId | UUID | Yes |
| learningType | Enum | Yes |
| title | String | Yes |
| progress | Decimal | No |
| status | Enum | Yes |
| notes | String | No |
| occurredAt | Timestamp | No |
| createdAt | Timestamp | Yes |
| updatedAt | Timestamp | Yes |
| archivedAt | Timestamp | No |

---

# Relationships

- Belongs to User
- Produces Learning Reports
- Produces Learning Statistics

---

# Constraints

- Identifier immutable.
- Title required.
- Archived records are read-only.

---

# Validation Rules

- Required fields cannot be empty.
- Progress must be between 0 and 100.
- Timestamp generated by the system.
