# Work Entity

**Document ID:** LOS-DAT-ENT-010

**Status:** Draft

**Owner:** Data Team

---

# Purpose

The Work Entity represents professional activities, attendance, projects, meetings, and career-related records managed by Life OS.

---

# Responsibilities

- Store work records.
- Track attendance.
- Track overtime.
- Track projects.
- Support productivity reporting.

---

# Attributes

| Name | Type | Required |
|------|------|----------|
| id | UUID | Yes |
| userId | UUID | Yes |
| workType | Enum | Yes |
| projectId | UUID | No |
| status | Enum | Yes |
| duration | Integer | No |
| notes | String | No |
| occurredAt | Timestamp | Yes |
| createdAt | Timestamp | Yes |
| updatedAt | Timestamp | Yes |
| archivedAt | Timestamp | No |

---

# Relationships

- Belongs to User
- References Projects
- Produces Work Reports
- Produces Productivity Statistics

---

# Constraints

- Identifier immutable.
- Work type required.
- Archived records are read-only.

---

# Validation Rules

- Required fields cannot be empty.
- Duration must be zero or greater.
- Timestamp generated by the system.
