# Health Entity

**Document ID:** LOS-DAT-ENT-008

**Status:** Draft

**Owner:** Data Team

---

# Purpose

The Health Entity represents health-related records managed by Life OS.

---

# Responsibilities

- Store health records.
- Store measurements.
- Track medications.
- Track supplements.
- Support health reporting.

---

# Attributes

| Name | Type | Required |
|------|------|----------|
| id | UUID | Yes |
| userId | UUID | Yes |
| recordType | Enum | Yes |
| value | Decimal | No |
| unit | String | No |
| notes | String | No |
| occurredAt | Timestamp | Yes |
| createdAt | Timestamp | Yes |
| updatedAt | Timestamp | Yes |
| archivedAt | Timestamp | No |

---

# Relationships

- Belongs to User
- Produces Health Reports
- Produces Health Statistics

---

# Constraints

- Identifier immutable.
- Record type required.
- Archived records are read-only.

---

# Validation Rules

- Required fields cannot be empty.
- Measurement units must be supported.
- Timestamp generated by the system.
