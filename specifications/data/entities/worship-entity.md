# Worship Entity

**Document ID:** LOS-DAT-ENT-009

**Status:** Draft

**Owner:** Data Team

---

# Purpose

The Worship Entity represents worship activities managed by Life OS.

---

# Responsibilities

- Store worship records.
- Track recurring worship.
- Support worship statistics.
- Support historical reporting.

---

# Attributes

| Name | Type | Required |
|------|------|----------|
| id | UUID | Yes |
| userId | UUID | Yes |
| activityType | Enum | Yes |
| status | Enum | Yes |
| value | Decimal | No |
| unit | String | No |
| notes | String | No |
| occurredAt | Timestamp | Yes |
| createdAt | Timestamp | Yes |
| updatedAt | Timestamp | Yes |
| archivedAt | Timestamp | No |

---

# Relationships

- Belongs to User
- Produces Worship Reports
- Produces Worship Statistics

---

# Constraints

- Identifier immutable.
- Activity type required.
- Archived records are read-only.

---

# Validation Rules

- Required fields cannot be empty.
- Activity type must be supported.
- Timestamp generated by the system.

---

# Security

- Accessible only by owner.
- Audit logging mandatory.
